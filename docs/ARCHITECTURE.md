## Components

- **lowfat CLI** (`crates/lowfat`) вҖ” clap entry point, dispatches subcommands.
- **lowfat Runner** (`crates/lowfat-runner`) вҖ” executes the real command, loads
  plugins via `HybridRunner`, and walks the pipeline stages.
- **Config** (`crates/lowfat-core`) вҖ” resolves `.lowfat` TOML + env vars into a
  `RunfConfig` (level, plugin dir, conditional pipelines).
- **Plugins** (`crates/lowfat-plugin`) вҖ” manifest + `.lf` DSL files. Bundled
  plugins are embedded via `include_str!`; user plugins live under
  `~/.lowfat/plugins/`. Disk wins on name collision.
- **Builtins** вҖ” in-process processors (`strip-ansi`, `head`, `grep`,
  `dedup-blank`, `normalize`, вҖҰ) used as pipeline stages.
- **SQLite metrics** вҖ” `~/.lowfat/data/` tracks token savings and invocation
  history (powers `lowfat stats` and `lowfat history`).
